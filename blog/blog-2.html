1. 作るアプリの概要

今回は、簡単なカウントアップアプリを作ります。

2. HTML & CSSの実装

<!DOCTYPE html>
<html lang="ja">
<head>
    <title>カウントアップアプリ</title>
</head>
<body>
    <p id="count">0</p>
    <button onclick="increment()">+1</button>
</body>
</html>

【ここに広告】

3. JavaScriptの処理

function increment() {
    let count = document.getElementById("count");
    count.textContent = Number(count.textContent) + 1;
}

4. まとめ

このアプリを改造して、ダークモードやローカルストレージ対応にするとさらに便利になります！
